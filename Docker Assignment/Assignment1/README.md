# Docker Assignment 1

## Part 1: Static Website Hosting via Docker

### Objective
Create a Dockerized utility to host a static website accessible via:

```text
maqbool.codeCatalyst.com
```

### Requirements

| Time Slot | Output |
|------------|---------|
| 10:00 AM - 12:00 PM | Hello from Maqbool |
| 04:00 PM - 06:00 PM | Hello from Buddy |
| Other Times | Website Closed |

### Test

```bash
curl http://localhost
```
